# Meridian Trade & Logistics — Supply Chain Delay Analysis

## Project Overview

Analysis of shipment delays across Southeast Asian maritime and air freight routes for Meridian Trade & Logistics Pte. Ltd. (Singapore).

Goal: identify root causes of delivery delays, build KPI dashboard for the COO.
